各種スクリプト名に「_AKIRA」とついているのは、これがついていないとスクリプト名が被ってしまうかもしれない、との懸念から作成者の名前を記載しております。  
それ以外の理由はなく、特に拘りなどもないため適宜変更していただいて構いません。    

**二次配布可、改変可、クレジット不要**
